# 筋肉マスター(Muscle Master)

## サイト概要
### サイトテーマ
経験者の知識とコツを共有することで、初心者の正しいフォーム習得をサポートするSNS。
​
### テーマを選んだ理由
私自身、筋トレを始めた当初は、自己流のフォームが正しいのか分からず、怪我を恐れて十分な負荷をかけられませんでした。この不安は、筋トレを挫折する大きな原因の一つになっています。
世の中には多くの筋トレ記録アプリやSNSが存在しますが、単に「重量と回数」を共有するだけでなく、「どの筋肉に効かせるかという意識ポイント」や「正しいフォームのコツ」といった、最も重要でありながら言語化が難しい知識を共有する場が不足していると感じました。
このアプリは、経験者の生きたノウハウを集約し、初心者が安全かつ効率的にトレーニングに取り組める環境を提供することで、筋トレ継続率の向上に貢献できると考え、このテーマで制作することにいたしました。

### ターゲットユーザ
- 筋トレを始めたばかりで、自分のフォームが正しいか不安な人。

- 自己流トレーニングに限界を感じ、特定の筋肉に効かせるためのコツを知りたい人。

- より効率的に筋肉を成長させるために、ベテラントレーニーの知識を参考にしたい人。

- 自分の経験や知識を共有することで、他者の成長をサポートしたい経験者。
​
### 主な利用シーン
- ジムで新しい種目を始める前に、正しいフォームや注意点を確認したい時。

- トレーニング中に、目標とする経験者の意識ポイントを再確認したい時。

- 自分のトレーニング動画をアップロードし、熟練者から具体的なフォーム修正のアドバイスをもらいたい時。

- 自分の知識を共有し、「参考になった」という評価を得て、貢献欲求を満たしたい時。
